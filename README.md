# 🤖 AI开发工具测评站

> 客观实测 · 多维度对比 · 选型指南 · 持续更新

🌐 **在线访问**：https://lanxinAIhub.github.io/ai-tools-review/

## 📋 简介

本项目对主流AI开发工具进行客观实测对比，包括：

| 类别 | 工具 |
|------|------|
| 🌎 国际模型 | Claude 3.5/4, GPT-4o, Gemini 2.0 |
| 🇨🇳 国产模型 | 百度文心, 阿里通义, 字节豆包, MiniMax |
| 🖥️ 本地模型 | Ollama, LM Studio, vLLM |
| 🔧 开发框架 | LangChain, LlamaIndex, AutoGPT |

## 📊 对比维度

- 💰 **价格** - 各平台官方定价（元/千Token）
- ⚡ **速度** - 首Token延迟实测
- 📈 **质量** - 基准测试分数（MMLU/HumanEval）
- 🎯 **易用性** - API复杂度/文档质量

## 📁 页面结构

- `/` - 首页：工具概览 + 推荐榜单
- `/tools/` - 工具详情页
- `/comparison/` - 多维度对比表格
- `/methodology/` - 评测方法论

## 🚀 GitHub Pages

本项目通过 GitHub Pages 托管，静态文件位于 `docs/` 目录。
