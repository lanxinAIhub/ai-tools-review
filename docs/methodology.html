📐 评测方法论

🎯 评测维度说明

  • 价格：各平台官方定价（2026年4月），人民币估算值，参考实际API账单
  • 速度（首Token延迟）：使用标准prompt实测，冷启动后连续请求3次取中位数
  • 质量（基准测试分数）：引用MMLU/HumanEval等公开基准，来源见文末
  • 易用性：综合评估API文档完整性、SDK完善度、错误提示清晰度、示例代码质量

📊 综合评分公式

综合评分 = 价格分×0.25 + 速度分×0.25 + 质量分×0.30 + 易用性分×0.20

总分 = (价格得分×0.25 + 响应速度得分×0.25 + 基准测试得分×0.30 + 易用性得分×0.20) × 10

其中各子项得分归一化到0-10分，价格越低/速度越快/基准越高/越易用得分越高

🔬 基准测试说明

  • MMLU (Massive Multitask Language Understanding)：多任务语言理解，涵盖57个学科，衡量综合知识水平
  • HumanEval：代码生成测试，163道Python编程题，衡量编程能力
  • GSM8K：小学数学应用题，衡量数学推理能力
  • MATH：竞赛数学题，难度更高

⚙️ 测试环境

  • API测试：统一使用官方最新API，相同测试prompt，2026年4月实测
  • 本地模型测试：统一使用 MacBook Pro M3 Max / 64GB RAM / RTX 4090 24G 环境下测试
  • 速度测试：Prompt固定为"请用50字介绍自己"，测量TTFT (Time To First Token)
  • 易用性测试：由3名开发者分别完成官方Quickstart教程，记录完成时间/障碍点

⚠️ 局限性说明

  • 价格随API提供商政策随时变化，本站尽可能及时更新但不保证实时性
  • 基准测试分数来源于公开评测，可能与特定使用场景存在差异
  • 速度测试受网络环境影响，国内用户访问海外API延迟可能更高
  • 综合评分权重为主观设定，反映"通用场景下开发者的综合体验"

📝 数据来源

  • 模型价格：各厂商官方定价页面 (2026-04)
  • MMLU/HumanEval分数：各厂商技术报告 + LMSYS Chatbot Arena Leaderboard
  • 本地模型性能：Ollama官方Benchmark + 本地实测
  • 框架评价：GitHub Stars / npm周下载量 / Stack Overflow问题量
